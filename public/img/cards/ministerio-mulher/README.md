# Imagens do Ministério da Mulher

Esta pasta deve conter as imagens utilizadas na página do Ministério da Mulher.

## Imagens Necessárias:

1. **mm_header.webp** - Imagem de cabeçalho da página (recomendado: 1920x300px)

2. **pesquisa01.webp** - Página 1 da pesquisa (informações pessoais)

3. **pesquisa02.webp** - Página 2 da pesquisa (áreas de interesse)

4. **pesquisa03.webp** - Página 3 da pesquisa (disponibilidade e talentos)

## Notas:
- As imagens devem estar em formato WebP para melhor performance
- Os arquivos `mm_header.webp` e `pesquisa0*.webp` no repositório são versões provisórias (geradas para o site funcionar)
- Substitua pelos arquivos finais quando tiver o PDF/design oficial da pesquisa
- A imagem de cabeçalho pode ser trocada por foto oficial do ministério (recomendado: 1920×300px)
